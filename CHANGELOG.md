# Changelog

## 0.1.0
- Implemented core `Security`, `Universe`, and `UniverseMember` models.
- Added OCC option resolver with packaged option-root rule data.
- Added in-memory and SQLite stores.
- Added `UniverseRegistry` with resolver-aware membership and include/exclude resolution.
- Added JSON and CSV universe member import/export.
- Added `securities` CLI.
- Added executable examples, tests, CI workflow, TestPyPI workflow, and PyPI
  release workflow.
