# Roadmap

## v0.1 Foundation

- `Security`, `Universe`, and `UniverseMember`
- `UniverseRegistry`
- in-memory store
- SQLite store
- OCC option resolver
- packaged index option rules
- packaged adjusted option rules
- user-supplied rule overrides
- CSV/JSON import/export
- `securities` CLI
- examples and tests

## v0.2

- expanded default option-root rules
- query helpers
- validation utilities
- SQLite migration helpers
- richer import/export schemas

## v0.3

- PostgreSQL store plugin
- audit history
- executable dynamic universe definitions

## Out of Scope

- broker integrations
- market-data integrations
- corporate-action feed processing
- order execution
- risk engine
- web UI
